## Notes on rendering architecture

- Content files use `dangerouslySetInnerHTML` inside `<script type="application/ld+json">` tags
- Dynamic import (`next/dynamic`) does NOT use `ssr: false`, so content is server-rendered
- CSP includes `'unsafe-inline'` in `script-src`, allowing inline JSON-LD
- Each blog page renders 7-8 JSON-LD blocks total (3 global org/website/software + Article + BreadcrumbList + Organization + FAQPage [+ HowTo where applicable])
